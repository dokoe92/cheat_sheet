HTML

Table tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head> The set of rows defining the cloumn headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container - Makes a new row.
<td> Table Data The table row container - Fill in Data.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Tag Name Description
<colspan> Column Span Defines how many columns a td element should span.
<rowspan> Row Span Defines how many rows a td element should span.

Git

Command Name Description
pwd Print Working Dictionary Shows in which dictionary you are currently working.
ls list Shows which documents are in this dictionary.
cd change dictionary Changes the dictionary to the the target
(a.e. cd dominik/DevProjects/cheat_sheet)
git init initialize Initialize a new Git repositry (current working dictionary)
git add add Adds files into the staging area.
git commit -m "" commit Commits files into the repository with a note
